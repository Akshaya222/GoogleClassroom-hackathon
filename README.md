## Steps to run the application
### To run server
1. Change directory to the Server directory: `$ cd Server`
2. `$ npm install`
3. `$ npx nodemon app.js`

### To run client
1. Change direcotory to the client directory: `$ cd client`
2. `$ npm install`
3. `$ npm start`
